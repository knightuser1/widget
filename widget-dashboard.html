📱 Install on Phone

Android — Add to Home Screen
1. Open your site in Chrome
2. Tap the ⋮ menu (top right)
3. Tap "Add to Home screen"
4. Tap Add — done! ✓
Android — Build Real APK (Free)
1. Go to gonative.io or webintoapp.com
2. Paste your GitHub Pages URL
3. Download the .apk file
4. Enable "Install unknown apps" in Android settings
5. Install the APK — works like a real app!
iPhone
1. Open in Safari
2. Tap the Share icon (box with arrow)
3. Tap "Add to Home Screen"
4. Tap Add ✓
Tip! Open this page in Chrome → tap ⋮ → "Add to Home Screen" to install as an app
